# Instrucciones

#### Backend

**Instrucciones**:
Preferiblemente, proceder desde un entorno virtual de Python (venv).

_Estando dentro de la carpeta `/backend`..._
1. Ejecutar desde la consola `pip install -r requirements.txt`, para instalar dependencias.
2. Ejecutar desde la consola `fastapi dev main.py`.
La API estaría lista y configurada.

#### Frontend
Se utilizó el template de Vite: `React Typescript + SWC`.
Para los gráficos y SVG, se usó `d3`.

**Instrucciones**:
_Estando dentro de la carpeta `/frontend`..._
1. Ejecutar desde la consola `npm install`, para instalar dependencias.
2. Ejecutar desde la consola `npm run dev`.
